github 配置

因为博客其实是部署在github上,所以需要在github上新建一个仓库。新建的时候 Repository name 名称必须是 xxx.github.io (xxx是你自己起的名字,项目部署成功后使用这个地址xxx.github.io即可访问）其他选项默认即可。仓库有了，剩下的就是怎么把博客放进去关联起来了。这就需要git了

自定义域名

博客搭建的差不多了，可是访问地址不那么独特，xxx.github.io,看起来狠low的样子，一点都不符合自己渣渣的身份，像什么https://is.lipper.online 这样看来来比lipper.github.io看起来要舒服多吧。不过这一步可能是要花钱的，先要买一个域名，像阿里云，腾讯云都可以买，也很方便。第一次买优惠还是很大的，也可以挑一个自己喜欢的域名。购买完域名后需要对其解析，例如我的域名lipper.online,我将默认值设置为is,所以到时访问地址就是(is.lipper.online),所以这块可以默认，可以自己设置，记录类型选择为CNAME,线路为默认,记录值为qifengle.github.io(这个域名就是配置前能直接访问到的域名地址)然后保存.过个几分钟一般就可以正常访问了.

tip:记住购买完域名后要实名认证，不然依旧无法使用

至此,hexo博客搭建至github pages就完成了
